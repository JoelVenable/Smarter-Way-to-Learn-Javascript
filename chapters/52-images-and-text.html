Let's give an image the option to disappear... First we'll give it an id.

Then we add an event handler.

Then we add a CSS class for invisibility.

The function called by the event handler adds the CSS "hidden" class, so it disappears using the 'display: none' property.

Some notes:

  • This is just a variation of things already learned. In this example we used the className property instead of the innerHTML or the value properties.
  • 'className' is in camelCase, not 'class'.
  • Assigning a class name REPLACES any existing classes declared in the existing HTML code. Perhaps we could use the addClass() method instead?
